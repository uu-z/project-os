# Doc Templates

Use these as the default table shapes for the control-plane docs.

## `CONTEXT.md`

| Section | Recommended structure |
|---|---|
| One sentence | repo/domain one-line definition |
| Core terms | glossary table |
| Avoid terms | anti-synonym table |
| Boundaries | boundary table |
| Stable invariants | invariant table |

## `KERNEL.md`

| Section | Recommended structure |
|---|---|
| One sentence | repo operating-kernel definition |
| Stable patterns | stable-pattern table |
| Candidate patterns | candidate table |
| Promotion upstream | promotion table |
| Rejected patterns | rejection table |
| Update protocol | write-rule table |

## `STRATEGY.md`

| Section | Recommended structure |
|---|---|
| One sentence | one-line definition |
| Goals | goal table |
| In scope / out of scope | scope table |
| Success definition | success criteria table |
| Non-goals | non-goal table |
| Hard rules | constraint table |

## `ARCHITECTURE.md`

| Section | Recommended structure |
|---|---|
| Mainline | system flow table or diagram |
| Boundaries | boundary table |
| Truth sources | truth-source table |
| Runtime ownership | runtime table |
| Anti-patterns | forbidden-pattern table |

## `ENGINEERING.md`

| Section | Recommended structure |
|---|---|
| Official commands | command table |
| Verification path | verification matrix |
| Artifact meanings | artifact table |
| Constraints | engineering rule table |
| Acceptance criteria | acceptance matrix |

## `PROJECT.md`

| Section | Recommended structure |
|---|---|
| Current stage | stage table |
| Milestones | milestone table |
| Owners | owner table |
| Timeline | weekly cadence table |
| Risks | risk register table |

## `BLOCKERS.md`

| Section | Recommended structure |
|---|---|
| P0 blockers | blocker table |
| P1 blockers | blocker table |
| Deferred noise | defer table |
| Dependencies | dependency table |

## `HARNESS.md`

| Section | Recommended structure |
|---|---|
| Read order | read-order table |

| Escalation | escalation table |
